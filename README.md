# kasus-investigator

Skill AI untuk investigasi kasus secara sistematis dan terstruktur.

## Fitur

- Investigasi multi-domain: OSINT, forensik digital, SNA, analisis data, keamanan siber
- Workflow 5 fase: Intake → Collection → Analysis → Verification → Reporting
- Analisis temporal, jaringan (SNA), sentimen, anomali, dan forensik
- Laporan investigasi otomatis dengan confidence scoring
- Support paralel untuk pengumpulan dan analisis data

## Instalasi

```bash
npx skills add InitialJawa/kasus-investigator
```

## Penggunaan

Aktifkan skill ini dengan:
- "Investigasi kasus ..."
- "Analisis pola ..."
- "Forensik ..."
- "Audit ..."
